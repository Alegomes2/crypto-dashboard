# 💰 Crypto Dashboard

Um dashboard interativo desenvolvido em React que exibe dados em tempo real de criptomoedas, com visual moderno e funcionalidades dinâmicas.

---

## 🚀 Demonstração

🔗 (https://crypto-dashboard-seven-jet.vercel.app/)

## 🧠 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar e demonstrar:

* Consumo de APIs externas
* Gerenciamento de estado com React
* Componentização
* Criação de interfaces modernas
* Manipulação de dados em tempo real

Os dados são fornecidos pela API da CoinGecko.

---

## ⚙️ Tecnologias utilizadas

* React
* Vite
* Axios
* Chart.js
* CSS

---

## ✨ Funcionalidades

* 🔍 Busca de criptomoedas em tempo real
* 💳 Exibição em cards interativos
* ⭐ Sistema de favoritos
* 📈 Visualização de gráfico ao clicar na moeda
* 🎨 Interface moderna e responsiva

---

## 🔌 Integração com API

O projeto utiliza a API pública da CoinGecko para obter:

* Lista de criptomoedas
* Preço atual
* Histórico de preços

---

## ▶️ Como rodar o projeto

```bash
# Clonar repositório
git clone https://github.com/SEU-USUARIO/crypto-dashboard.git

# Entrar na pasta
cd crypto-dashboard

# Instalar dependências
npm install

# Rodar projeto
npm run dev
```
